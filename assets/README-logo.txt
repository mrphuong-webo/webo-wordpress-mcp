(Template hướng dẫn upload logo/icon/banner cho WordPress.org)

Bạn cần upload file:
- icon-128x128.png (icon plugin, PNG nền trong suốt, 128x128)
- banner-1544x500.png (banner plugin, PNG, 1544x500)

Vào thư mục assets/ này để plugin hiển thị đúng trên WordPress.org.

Ví dụ nhúng logo vào README.md:

![Plugin Icon](assets/icon-128x128.png)
![Plugin Banner](assets/banner-1544x500.png)

Nếu cần chuyển đổi định dạng/kích thước, hãy gửi file hoặc yêu cầu.
